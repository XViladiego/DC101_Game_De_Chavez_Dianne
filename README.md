# Catch Love Game ❤️

## Description
Catch Love Game is a browser-based reflex game where the player must catch falling love-related emojis such as hearts, kisses, and roses.  
The goal is to collect as many love items as possible while avoiding broken or negative love symbols. Catching a broken heart results in a game over.

## Game Mechanics
- Love emojis increase the score
- Broken or negative love emojis end the game
- Items fall randomly from the top of the screen
- The game becomes more challenging as you play

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run the Game
1. Download or clone the repository
2. Open the `index.html` file in any modern web browser
3. Click on the falling emojis to play

## Controls
- Mouse Click / Touch: Catch falling emojis
